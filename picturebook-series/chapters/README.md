# 八十一章章节稿

本目录存放《道德经》八十一章逐章绘本材料。每个章节文件都先按统一模板留位，后续逐章补充。

章节总表见 [chapter-index.md](chapter-index.md)。

## 每章固定结构

每章只保留四个部分，写成纯文字稿，不使用表格，不写画面描述。每章分为 15 页：

1. 原文：放本章《道德经》原文。
2. 释义：用孩子能理解的话解释。
3. 正文：用故事、典故或讲述展开。
4. 回扣：回到原文，点明故事和章句的关系。

## 使用方式

- 新写一章前，先复制 `_chapter-template.md` 的结构到对应 `ch###.md`。
- 找不到故事时，也可以把典故、案例或概念解释写进“正文”部分。
- 每章完成后，在 `summaries/` 建立摘要，并更新 `RECENT_EVENTS.md`。
