# Research

这里存放每章可用的资料、典故、历史案例、现代案例和全球故事素材。

## 子文件

- `source-log.md`：资料来源登记。
- `story-candidate-index.md`：故事候选索引，按章节和主题归档。
- `china-ancient.md`：中国古代典故与人物故事。
- `china-modern.md`：中国现代案例。
- `usa-and-world.md`：美国与世界案例。
- `motif-index.md`：水、木、门、路、谷、婴儿等意象索引。

## 资料状态标记

- `fact`：可核查事实。
- `legend`：传说或传统说法。
- `interpretation`：学者解释、注本解释或作者理解。
- `fiction`：为绘本原创或明显改编的故事。
- `candidate`：暂未核查，不进入正文。
