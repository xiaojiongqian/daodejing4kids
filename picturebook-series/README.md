# 《道德经》儿童绘本系列项目

这个目录用于承载基于《道德经》的系列儿童绘本材料。项目分为两大部分：

1. 背景篇：讲老子、《道德经》的形成、时代背景、文本流传和相关传说。
2. 章节篇：按八十一章推进；每章包含原文、儿童可理解的含义、相关典故与故事素材、绘本页脚本和画面提示。

## 目录结构

- `INDEX.md`：项目总索引与当前主线。
- `CURRENT_STATE.md`：当前写作状态。
- `ARC_STATUS.md`：整体卷册与章节进度。
- `OPEN_LOOPS.md`：尚未解决的选题、设定和资料问题。
- `FORESHADOWS.md`：跨章反复出现的图像、物件和主题伏笔。
- `WORLD.md`：世界观与历史背景。
- `CHARACTERS.md`：固定角色、讲述者和候选人物。
- `CHARACTER_ARCS.md`：角色成长线。
- `THEMES.md`：核心主题与反复出现的哲理。
- `STYLE_GUIDE.md`：语言、年龄段、绘本节奏和禁区。
- `RULES.md`：资料使用、改写边界和事实核查规则。
- `RECENT_EVENTS.md`：最近完成章节的可继承记忆。
- `background/`：背景篇材料。
- `chapters/`：八十一章章节稿。
- `research/`：典故、历史资料、现代案例和全球故事素材。
- `visual/`：画面风格、角色造型和分镜模板。
- `summaries/`：章节摘要与卷册摘要。
- `production/`：出版、排版、审校和待办清单。

## 当前原则

- 只把用户已经明确的创作方向写成稳定项目事实。
- 章节具体故事、典故和现代案例先作为 `candidate`，经资料核查与创作确认后再进入正文。
- 每章优先形成一个孩子能记住的核心画面或故事动作，而不是只堆解释。
- 原文、释义、故事、画面四层分开放，便于后续审校和改写。
