# 背景篇

背景篇用于放在八十一章之前，帮助儿童理解老子和《道德经》从哪里来、为什么重要、有哪些传说和版本故事。

## 建议章节

1. 老子是谁：历史人物与传说人物。
2. 老子生活的时代：周王室、诸侯、礼法、战争和思想家。
3. 《道德经》是什么：短短五千言，为什么能流传两千多年。
4. 出关传说：老子、青牛、函谷关和尹喜。
5. 文本流传：传世本、王弼本、河上公本、马王堆帛书、郭店楚简。
6. 怎么读《道德经》：它不是谜语，也不是命令，而是一种观察世界的方法。

## 写作层次

- `事实层`：可核查的历史和文本资料。
- `传说层`：民间与文献中的故事，需要标明“传说”。
- `儿童解释层`：把背景讲成孩子能理解的情境。
- `绘本画面层`：每节对应的画面和分镜。
