# Open Loops

- id: loop-age-band
  label: 目标读者年龄段
  type: task
  introduced_in: project setup
  current_status: active
  stakes: 年龄段会影响字数、语言难度、画面密度和哲理解释方式。
  owners: author
  expected_payoff_window: 背景篇开写前
  latest_update: 待确认。

- id: loop-narrative-frame
  label: 系列叙事框架
  type: promise
  introduced_in: project setup
  current_status: active
  stakes: 是否有固定角色会影响八十一章的连续性和儿童代入感。
  owners: author
  expected_payoff_window: 第一章样章前
  latest_update: 可选“讲述型”“空空村系列故事型”“师徒问答型”“每章独立寓言型”。

- id: loop-source-base
  label: 原文底本与异文处理
  type: task
  introduced_in: project setup
  current_status: active
  stakes: 决定每章原文如何呈现，以及是否加入简帛异文说明。
  owners: researcher
  expected_payoff_window: 章节篇正式启动前
  latest_update: 当前 `reference/` 已有传世本选篇和版本笔记。

- id: loop-story-research
  label: 每章故事素材搜集
  type: task
  introduced_in: project setup
  current_status: active
  stakes: 影响章节是否能形成生动、可信、跨文化的故事解释。
  owners: researcher / author
  expected_payoff_window: 每章开写前
  latest_update: 已建立 `research/` 入口。
