# Background

本目录存放全球版背景篇材料，目标是帮助不熟悉中国古典思想的读者进入《道德经》，同时不给章节正文增加过重注释。

## Planned Topics

- 老子其人与传说：区分可考史料、传统叙事和后世想象。
- 《道德经》的文本形成：传世本、帛书、郭店、章序和异文。
- 主要注释传统：王弼、河上公、后世道教与现代解释。
- 英译与全球接受：Legge 等早期译本，现代 `Dao/Tao` 译名问题。
- 阅读方法：把《道德经》当作诗、哲学、政治文本、修养文本和视觉文本时的差异。
