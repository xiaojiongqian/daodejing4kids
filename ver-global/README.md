# 《道德经》全球版绘本项目

这个目录用于承载面向全球读者的《道德经》绘本脚本资料。这里的“全球版”默认不是儿童版，而是给成人普通读者、艺术书读者、跨文化哲学读者看的图文作品底稿。

项目先沿用 `ver-for-kids/` 已验证的工作方式：把原文、释义、创作素材、故事脚本、视觉提示和制作备忘分开管理；但语言、叙事密度和视觉气质会重新面向成人与国际语境设计。

## 目录结构

- `INDEX.md`：项目总索引与当前主线。
- `CURRENT_STATE.md`：当前写作状态。
- `ARC_STATUS.md`：整体进度与阶段目标。
- `OPEN_LOOPS.md`：尚未决策的问题。
- `RULES.md`：资料、翻译、改写和跨文化表达规则。
- `STYLE_GUIDE.md`：全球版文字、叙事和视觉语气。
- `THEMES.md`：全书核心主题。
- `WORLD.md`：全球版视觉世界与叙事空间。
- `CHARACTERS.md`：可复用人物/叙事视角候选。
- `CHARACTER_ARCS.md`：若使用连续人物，其内在线索记录。
- `FORESHADOWS.md`：跨章图像、物件与哲学伏笔。
- `RECENT_EVENTS.md`：最近完成工作的可继承记忆。
- `background/`：老子、《道德经》、文本流传与全球接受背景。
- `chapters/`：八十一章逐章材料与绘本脚本。
- `research/`：跨文化典故、历史案例、现代图像和来源登记。
- `visual/`：艺术方向、页面模板、复现图像系统。
- `summaries/`：章节摘要与卷册摘要。
- `production/`：编辑、排版、审校和制作待办。

## 当前原则

- 原文入口稳定，解释保持开放：不把任何单一注本讲成唯一答案。
- 全球化不是把《道德经》改成西方格言，而是让读者在准确语境中进入它。
- 绘本可以诗性、简洁、视觉化，但不得牺牲义理精度。
- 每章可以由一篇到多篇构成；第一篇讲原文和整体义理，后续篇补典故、故事、历史考据与哲学升华。
- 每篇章节文件都应是可直接复制给绘本生成 agent 的干净页面稿；画面提示、制作限制和来源说明放入辅助文件，不混入正文。
- 每章优先形成能被图像承载的核心经验，而不是只做概念讲解。
