# Page Layout Template

## Single Chapter Suggested Structure

- Opening threshold: 以一个可视物件打开本章核心问题。
- Source echo: 让本章原文或核心句短暂出现。
- Concrete encounter: 人、物、场景发生变化。
- Philosophical turn: 画面让读者意识到原先理解不够。
- Return: 回到章义，但不把结尾写成口号。

## Text Length

- English: usually 1-3 short sentences per page.
- Chinese working draft: usually 1-3 short sentences per page.
- If a page carries original source text, surrounding prose should be lighter.
