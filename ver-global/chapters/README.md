# Chapters

本目录存放全球版《道德经》八十一章逐章材料与绘本脚本。

## Chapter / Part Structure

全球版每章可以由一篇到多篇构成，统一使用分篇文件命名：

- `ch001-1.md`：第一篇，必须完整呈现本章原文、整体义理和主故事入口。
- `ch001-2.md`、`ch001-3.md`：后续篇，可补充典故、历史考据、跨文化故事，并做论述升华。

章节文件本身只放正文页稿，不放 metadata、生成提示、视觉说明或研究说明。每篇建议包含：

1. 中文页稿：`原文 / 释义 / 术语与考据简注 / 正文 / 回扣`。
2. 英文页稿：`Original Text / Interpretation / Key Terms and Notes / Main Text / Return to the Original`。
3. 后续典故篇开头应直接进入新的典故、历史考据、人物场景或论述推进，不要先重复第一篇已经呈现的原文和释义；需要回扣原文时，放在正文中段或末尾。

第一篇通常包括原文和整体解释；后续篇不必重复全章原文，但必须明确回扣本章章句。

## Page Count

- 每篇默认试稿范围：12-20 页。
- 第 1 章当前拆为 3 篇，每篇 17 页。
- 若某章哲学或典故密度更高，可以拆为更多篇；不要把互不相干的故事硬塞进同一篇。
- 全球版不同于儿童版，每页可以容纳更厚的文字与思辨。单页不必严格限制为 2-3 句；当页内需要完成考据、哲学辨析或叙事转折时，可以写成 1-3 个较完整段落。
- 增加文字不是堆解释。每页仍应有一个明确的画面中心和一个明确的思想推进。

## Language Rule

- 中文原文照项目底本。
- Legge 译文作为英文原文入口时照录，不现代化改写。
- 英文脚本文字为原创，使用现代全球读者更容易进入的表达。

## Files

- `_chapter-template.md`：章节模板。
- `chapter-index.md`：章节索引。
- `ch001-1.md`、`ch001-2.md`、`ch001-3.md`：第一章三篇可生产稿。
