# Open Loops

- id: loop-primary-language
  label: 全球版主语言
  type: decision
  introduced_in: global setup
  current_status: active
  stakes: 决定章节文件中中文工作稿、英文成稿和注释的比例。
  owners: author
  expected_payoff_window: 第一章三篇可生产稿审阅后
  latest_update: 当前样章采用中文工作稿 + English Global Script。

- id: loop-format
  label: 出版与展示形态
  type: decision
  introduced_in: global setup
  current_status: active
  stakes: 成人绘本、艺术书、网页互动版和展览文本对页数、注释和视觉节奏要求不同。
  owners: author / production
  expected_payoff_window: 前三章样章完成前
  latest_update: 暂按成人绘本/艺术书脚本组织。

- id: loop-narrative-frame
  label: 连续人物或独立寓言
  type: decision
  introduced_in: global setup
  current_status: active
  stakes: 影响八十一章是否有角色成长线、重复场景和叙事连续性。
  owners: author
  expected_payoff_window: 第 1-3 章试稿后
  latest_update: 第 1 章使用“旅行者/向导”作为非锁定框架。

- id: loop-source-apparatus
  label: 异文与注释呈现
  type: research
  introduced_in: global setup
  current_status: active
  stakes: 全球读者需要信任来源，但正文又不能被注释压住。
  owners: researcher / editor
  expected_payoff_window: 背景篇开写前
  latest_update: 章节文件只列必要来源；深入版本说明放入 `background/` 或附录。

- id: loop-visual-style
  label: 视觉风格锁定
  type: design
  introduced_in: global setup
  current_status: active
  stakes: 决定画面是水墨现代化、全球当代插画、摄影拼贴，还是更抽象的版画语言。
  owners: art director
  expected_payoff_window: 第 1 章视觉样页前
  latest_update: 暂定“留白、物质质感、世界地图/自然地貌/手写文字”的混合方向。
