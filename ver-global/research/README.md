# Research

本目录存放全球版每章可用的资料、典故、历史案例、现代案例、视觉参照和来源登记。

## 子文件

- `source-log.md`：资料来源登记。
- `story-candidate-index.md`：故事/类比候选索引，按章节和主题归档。
- `motif-index.md`：门、路、水、谷、器、空白、地图等意象索引。

## 资料状态标记

- `primary`: 原文、古注、版本资料等一级文本。
- `commentary`: 注释、解释、学者观点或项目内部义理判断。
- `fact`: 可核查历史事实。
- `legend`: 传说或传统说法。
- `fiction`: 为绘本原创或明显改编的叙事。
- `candidate`: 暂未核查，不进入最终正文。

## 使用原则

- 每章写作前先登记关键来源。
- 新引入的跨文化桥梁要说明“帮助理解什么”和“可能造成什么误读”。
- 可靠但不适合正文的材料，可以保留为创作者背景，不必硬塞进页面脚本。
