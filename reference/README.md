# 道德经资料索引

这个目录用于存放《道德经》相关的经典资料、版本线索、注本笔记和主题索引。当前先收集一批可继续扩展的基础材料，后续可按“原文校勘、历代注本、现代研究、儿童改写素材”继续细分。

## 已收录

- [传世本选篇](texts/dao-de-jing-selected.md)：先收录常用传世本若干章，便于引用、改写和主题归纳。
- [版本与出土文献笔记](versions/version-notes.md)：梳理王弼本、河上公本、马王堆帛书、郭店楚简等版本线索。
- [王弼与河上公注本笔记](commentaries/wangbi-heshanggong.md)：记录两类经典注释传统的阅读重点。
- [核心概念索引](themes/core-concepts.md)：按“道、德、无为、自然、柔弱、水、小国寡民”等主题归纳可用材料。
- [参考来源](bibliography.md)：记录在线原典、资料库和后续建议查阅书目。

## 使用原则

- 古籍原文宜保留异文空间，不急于统一成唯一“标准答案”。
- 引用具体章句时，尽量标明所据版本；如果用于正式出版或课程资料，应再查纸本校勘本。
- 现代网页上的整理、标点、解释可能有版权或校勘差异，除少量摘引外，建议只记录来源链接和自己的读书笔记。
- 儿童故事、绘本、讲解稿可从这里提取主题，但不要把注本解释直接当作唯一解读。
