# 道德经资料索引

本目录收录《道德经》相关的权威原文、历代注本、版本校勘、主题索引与儿童改写素材，供本"道德经儿童绘本"项目引用、讲解与改写之用。

资料分两类，处理方式不同：

- **古籍原文（公有领域）**：经文、王弼注、河上公章句等，**全文收录**，并经多源交叉校订（见下"文本来源与整理方法"）。
- **现代著作（受版权保护）**：南怀瑾、马保平等今人讲解，**只做导读/读书笔记 + 来源线索**，不转录原文（见"使用原则"）。

## 文本来源与整理方法（去伪存真）

原文部分不凭记忆录入，而由脚本从权威电子底本确定性生成，可复现、可审计：

1. **底本与参校本**（取材日期 2026-06-16）：
   - 维基文库《老子道德經注》（王弼注本，华亭张氏原本）—— 王弼本经文 + 王弼注。
   - 维基文库《老子河上公章句》（道经/德经）—— 河上公本经文 + 章题 + 注。
   - 中国哲学书电子化计划 ctext.org `ctp:dao-de-jing` —— 通行本经文（参校）。
2. **繁→简**：经 OpenCC `t2s` 确定性转换，并对其产生的少数生僻字形（如 繟、绵、稽、妖）做了可读化规范。
3. **交叉校订**：全经八十一章中，王弼本与 ctext 通行本经文有 **49 章完全一致**；其余各章异文、以及 6 处依通行本的校改，逐条记入 [版本异文校记](versions/variant-apparatus.md)。ctext 电子本个别处有脱误（如第 3 章），故以转写更严谨的王弼本为底本。
4. **校验**：生成文件经"与底本逐字比对"与"独立通读校对"两重检查，确认八十一章齐全、经注不错位、无残留标记。

> 严肃出版或课程使用时，仍应再核纸本校勘本（楼宇烈《王弼集校释》、高明《帛书老子校注》、朱谦之《老子校释》等）。

## 使用原则

- 古籍原文宜保留异文空间，不急于统一成唯一"标准答案"；引用具体章句时尽量标明所据版本。
- **现代著作有版权**：南怀瑾、马保平等今人讲解，本目录只记录导读笔记、来源线索与合法获取途径，除极少量明确标注的示例性引用外，不转录原文。
- 儿童故事、绘本、讲解稿可从这里提取主题，但不要把任何一家注本解释直接当作唯一解读；为想象与改写留白。
