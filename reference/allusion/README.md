## 体例（每条典故的字段）

每则典故为一个二级标题「## 序号. 标题」，下列要点：

- **出处**：可考的古籍原始出处；不确定者注「待考」并附线索。
- **关联**：该典故所印证的《老子》章次与经文/主题；同一典故跨多章出现者，并列全部章节。
- **典故**：用规范白话**复述**这则公有领域的史实/寓言（已在 OCR 文本基础上润色、纠正明显错字）。
- **南师讲法**：南怀瑾如何借此印证老子义理（**转述**，非照录原文）。
- **绘本用法**：面向儿童绘本创作的一句提示。

## 整理方法

由多智能体流水线对底本逐块**穷尽式**抽取，再经"完备性复检"二次扫描补漏，最后按"同一典故"归并去重（跨章合并为一条）、分类成文。共抽得约 740 条原始记录，归并后得 722 条。
